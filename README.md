# Claude AI Products & Certifications Portal

A comprehensive, beautifully designed portal for Claude AI products, documentation, learning resources, certifications, and interview preparation.

## 🌟 Features

- **Modern, Vibrant UI** - Beautiful gradient design with glassmorphism effects
- **Tab-Based Navigation** - Easy access to 8 different sections
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Interactive Elements** - Accordion Q&A, filterable content, hover effects
- **All-in-One Resource** - Everything you need about Claude AI in one place

## 🚀 Getting Started

### View Online
The portal is hosted and can be accessed directly at:
https://srinipusuluri.github.io/claude-learning-portal/

### Local Development
1. Clone the repository:
```bash
git clone https://github.com/srinipusuluri/claude-learning-portal.git
cd claude-learning-portal
```

2. Open `docs/index.html` in your browser, or serve it locally:
```bash
# Using Python
python -m http.server 8000 -d docs

# Using Node.js
npx serve docs
```

3. Visit `http://localhost:8000` in your browser

## 🛠️ Technical Details

- **Pure HTML/CSS/JavaScript** - No frameworks required
- **Single Page Application** - Fast, smooth navigation
- **CSS Variables** - Consistent theming with vibrant color system
- **Responsive Design** - Mobile-first approach
- **Accessibility** - Semantic HTML and ARIA attributes

## 🎨 Design Features

- **Color System**: Vibrant indigo/purple primary with pink, cyan, orange, and green accents
- **Typography**: Inter font family with Fira Code for code snippets
- **Animations**: Smooth transitions, hover effects, floating background orbs
- **Glassmorphism**: Frosted glass effects on tab bar and footer
- **Custom Scrollbar**: Styled to match the theme
